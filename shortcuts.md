### To create duplicate code line above/below: 
- Alt  +Shift + up/down arrow

### To wrap selected text in HTML tags
- Alt + W
    - This shortcut is provided by the htmltagwrap extension

### Toggle terminal window
- While terminal window is selected:  Ctrl + `

### Go to specific line
- Ctrl + G followed by line number

### Comment out the current line
- Ctrl + /

### Delete current line
- Ctrl + Shift + K

### Bring up a window with all VSCode keyboard shortcuts
- Ctrl + K then Ctrl + S
